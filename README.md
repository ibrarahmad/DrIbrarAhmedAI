## How to use this repo

```bash
git clone https://github.com/ibrarahmad/DrIbrarAhmedAI
cd DrIbrarAhmedAI/<build-folder>
# read that folder's README, then run the scripts in order
```

Each build folder is self-contained and has its own README with prerequisites,
step-by-step scripts, and a security checklist.

---

## A note on safety

These projects put capable agents and tools on real hosts. **Demo fast, secure
faster.** Don't hand an agent full tools until logs, allowlists, and approvals
are working. Each folder's README spells out the hardening steps — follow them.
